3. 1번 2번 조합 (가장 기본적인 모습)
```
function sum(a: number, b: number): number{
    return a + b;
}

sum(1, 2); // 3
sum(1, 2, 3); // 파라미터 개수가 일치하지 않아 에러 발생
```
